# 一个基于 module federation 的子应用集合 

## 基座地址
### management-frontend next 模板：
https://github.com/Lemenxzy/next-webpack-5-module-federation

## 运行

### demo1
```bash
cd son-module-federation
```

```bash
npm i
```

```bash
npm start
```

### 预览集成
```bash
npm run build
```

```bash
npm i http-server -g
```

```bash
http-server dist -p 3001
```
